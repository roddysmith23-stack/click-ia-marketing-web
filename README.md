# CLICK.IA Marketing Website

This is the official marketing landing page for CLICK.IA, built with React, Vite, and Tailwind CSS.

## Features
- **Hero Section**: High-impact value proposition with neon aesthetics.
- **How It Works**: 3-step visual flow.
- **Nano Studio Showcase**: Highlighting premium features.
- **Performance**: ROI data visualization.
- **Pricing**: 5-tier subscription table.

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment on Vercel

1. Make sure you have the Vercel CLI installed or connect your GitHub repository to Vercel.
2. The project includes a `vercel.json` formatted for Single Page Applications (SPA).
3. To deploy manually via CLI:
   ```bash
   vercel
   ```
4. Follow the prompts. The build command is `npm run build` and the output directory is `dist`.
